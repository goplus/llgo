## LLGo Plugin of LLDB

### Build with debug info

```shell
LLGO_DEBUG_SYMBOLS=1 llgo build -o cl/_testdata/debug/out ./cl/_testdata/debug
```

### Debug with lldb

```shell
_lldb/runlldb.sh ./cl/_testdata/debug/out
```
